# JHDanmakuRender

一个ios和osx通用的弹幕渲染引擎


## 介绍
因为对功能的需要 最终还是自己动手写了这个弹幕引擎 部分源码参考了[BarrageRenderer](https://github.com/unash/BarrageRenderer) 和[Bilibili Mac Client](https://github.com/typcn/bilibili-mac-client) 

demo基本涵盖了常用的功能 需要的看demo就行

* 支持全局的字体样式更改
* 支持实时回退功能
* 更简单的api
* 支持ios、osx系统
